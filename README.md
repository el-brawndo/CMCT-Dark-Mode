Context Menu Close Tab Dark Mode


Add a Close Tab menu item to the right-click context menus on all pages, with an icon suitable for users of Firefox UI CSS customizations that implement a dark context menu..


This extension is a modification of the Firefox addon "Context Menu Close Tab" by Steve Sharp: 

https://addons.mozilla.org/en-US/firefox/addon/context-menu-close-tab/


In turn, the original Firefox extension is a modification of the Chrome extension "Close Tab in Context Menu" by Daniel Nishi:

https://chrome.google.com/webstore/detail/close-tab-in-context-menu/lkjkhomjgdhefemghlimjkjdaibnnocg
